PublicApi (api in module ui.grid.cellNav )

Description

Public Api for cellNav feature

Methods

  • getCurrentSelection()

    returns an array containing the current selection
    array is empty if no selection has occurred

  • getFocusedCell()

    returns the current (or last if Grid does not have focus) focused row and column
    value is null if no selection has occurred

  • rowColSelectIndex(rowCol)

    returns the index in the order in which the RowCol was selected, returns -1 if the RowCol isn't selected

    Parameters

    • rowCol – {object} –

      the rowCol to evaluate

  • scrollToFocus(rowEntity, colDef)

    brings the specified row and column into view, and sets focus to that cell

    Parameters

    • rowEntity – {object} –

      gridOptions.data[] array instance to make visible and set focus

    • colDef – {object} –

      to make visible and set focus

    Returns

    {promise} –

    a promise that is resolved after any scrolling is finished

Events

  • navigate

    raised when the active cell is changed 

         gridApi.cellNav.on.navigate(scope,function(newRowcol, oldRowCol){})

    Parameters

    • newRowCol – {object} –

      new position

    • oldRowCol – {object} –

      old position

  • viewPortKeyDown

    is raised when the viewPort receives a keyDown event. Cells never get focus in uiGrid due to the difficulties of setting focus on a cell that is not visible in the viewport. Use this event whenever you need a keydown event on a cell

    Parameters

    • event – {object} –

      keydown event

    • rowCol – {object} –

      current rowCol position
