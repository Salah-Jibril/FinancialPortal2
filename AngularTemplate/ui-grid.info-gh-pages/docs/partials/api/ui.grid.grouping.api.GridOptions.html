GridOptions (api in module ui.grid.grouping )

Description

GridOptions for grouping feature, these are available to be set using the ui-grid gridOptions

Properties

  • enableGrouping

    Enable row grouping for entire grid.
    Defaults to true

  • groupingIndent

    Number of pixels of indent for the icon at each grouping level, wider indents are visually more pleasing, but will make the group row header wider
    Defaults to 10

  • groupingRowHeaderAlwaysVisible

    forces the groupRowHeader to always be present, even if nothing is grouped. In some situations this may be preferable to having the groupHeader come and go
    Defaults to false

  • groupingRowHeaderWidth

    Base width of the grouping header, provides for a single level of grouping. This is incremented by groupingIndent for each extra level
    Defaults to 30

  • groupingShowCounts

    shows counts on the groupHeader rows
    Defaults to true
